Integrate Onepay paygate into WooCommerce.

== Description ==

It allows people who use WooCommerce can payment via Onepay paygate.

== Installation ==

You can install the plugin by two ways:

1. Through Wordpress admin panel:
	- Goto **Plugins -> Add new**. Find **Woo Onepay** plugin into search box.
	- Then click **Install Now** button to active it.
	* if you have one, just click **Upload plugin** on the top and choose the plugin zip file.
2. Upload through FTP
	- Upload entire plugin folder into `/wp-content/plugins/` directory.
	- Active it in admin.
